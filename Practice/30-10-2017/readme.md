# Указания к практике

_**Перед началом практики убедитесь, что у вас установлена Node.js.**_

### Пошаговая инструкция

- Скачайте `dist.zip` и распакуйте его в какую-нибудь папку
- Напишите ваш код в `index.html`, включив в тег `<head>` следующий код:
  
  ```html
  <script>
      { data }
  </script>
  ```

- Откройте командную строку
  
  Если вы на Windows, ищите в списке установленных приложений
  `Node.js Command Prompt`.

  Если вы на macOS / Linux / etc. &mdash; запускайте любой терминал по вашему вкусу.

- Перейдите в папку с распакованным `dist.zip`
- Запустите сервер командой `node server.js`
- В браузере страница будет доступна как `localhost:8080`

# DOM API Micro Reference

## document.createElement(*tag* [, options])

Возвращает объект класса *HTMLElement*, представляющий новый элемент тега *tag*,
но не встроенный в DOM-дерево.

- `tag` &mdash; имя тега элемента, который будет создан, без угловых скобок
- `options` &mdash; объект, имена свойств которого станут атрибутами элемента,
  а значения свойств &mdash; значениями атрибутов (необязательный параметр)

## document.querySelector(*query*)

Возвращает *HTMLElement*, соответсвующий первому элементу в DOM-дереве,
удовлетворяющему *query*.

- `query` &mdash; строка-селектор (__*как в css*__)

### В следующий методах `$` является сокращением для объекта элемента.

## $.textContent

Строковое свойство, описывающее текстовое содержимое элемента.
Может быть изменено.

## $.classList.add(*classname*)

Добавляет *classname* к классам элемента.

- `classname` &mdash; строковое значение названия класса

## $.appendChild(*element*)

Добавляет *element* как дочерний элемент к *$*.

- `element` &mdash; объект типа HTMLElement, который будет дочерним для $.
